赛题背景

CAD 模型是现代工业设计的核心数字资产，广泛服务于云端协同、资产检索与数字孪生等场景。然而，高保真 CAD 模型包含海量参数化曲面与复杂拓扑，存储与传输代价高昂。如何在极低存储维度下保留其关键结构化信息，已成为三维智能设计规模化落地的关键瓶颈。

本次竞赛聚焦 CAD 模型中承载拓扑骨架的 曲线图 (Curve Graph) 结构，邀请全球三维视觉与几何深度学习的研究者，设计端到端的 神经编解码器（encoder-decoder） ，在隐空间最多 512 个 float32 值 的硬约束下，实现对曲线图几何与拓扑的高保真压缩与重建。我们期待由此涌现的新方法，能为云端 CAD 资产库、工业元宇宙与三维 AIGC 等下一代应用提供紧凑而结构化的表示基石。

赛题描述

曲线图结构

CAD 模型的边界表征 (B-rep) 包含参数化曲面 (Surface)、曲线 (Curve) 以及 vertex-edge-face 之间的拓扑关系。 其中，以曲线为边、以曲线端点/交汇点为节点所构成的曲线图 (Curve Graph) 是 B-rep 拓扑结构的核心骨架。 不同于非结构化的三角网格或点云，曲线图以极为紧凑的方式编码了 CAD 模型的拓扑与几何信息：

  • 节点 (Vertex) — 表示曲线的端点或多条曲线的交汇点，携带三维空间坐标
  • 边 (Edge / Curve) — 表示连接两个节点的曲线段，携带曲线的几何参数（曲线类型、控制点、权重等）或离散采样点序列
  • 邻接关系 (Adjacency) — 编码曲线之间的拓扑连接：哪些曲线共享端点、哪些曲线围成同一个面

任务定义

本赛题聚焦于 B-rep 中的曲线图结构（不涉及曲面的压缩），要求选手设计基于深度神经网络的曲线图编解码器 (Curve Graph Codec)：

  • 输入 — 完整的 CAD 曲线图 G = (V, E, A)
  • 编码 — 压缩为紧凑的隐变量表示
  • 硬约束 — 隐向量长度 ≤ 512 个 float32（超出直接记 0 分）
  • 解码 — 从隐变量中重建曲线图的几何与拓扑

核心挑战

  • 几何层面 — 每条曲线的形状需精确还原，微小的偏差会导致后续曲面重建失败
  • 拓扑层面 — 曲线间的连接关系必须严格保持，错误的邻接会导致模型边界出现缝隙、多余交叉或面片丢失，彻底破坏 CAD 模型的工程可用性
